1.set 基本用法

  • 数组去重 let newArr = [...new Set(targetArr)]
  • 字符串去重 let newStr = [...new Set(targetStr)].join('')
  • 添加的值类型不会转变

2.Set 实例的属性和方法

  • size:成员个数
  • add(val):添加
  • delete(val):删除
  • has(val):是否存在,返回boolean
  • clear()：清除所有成员
  • const array = Array.from(set):Set转数组
  • Set键值和键名相同
  • keys():键名遍历
  • values():键值遍历
  • entries():键值和键名遍历
  • Set filter 实现 并集 交集 差集

3.WeakSet

  • 只能存对象，弱引用，不可遍历
  • add(val):添加
  • delete(val):删除
  • has(val):是否存在,返回boolean
  • 适合存DOM节点
