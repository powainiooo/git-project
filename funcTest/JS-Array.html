Methods of Array

  • 1.contact(), array1.contact(array2, array3)

    1. 链接多个数组
    2. 不会改变现有数组，只返回被连接数组的副本
    let [arr0, arr1, arr2] = [[1,2], [3,4], [5,6]]
    let arr3 = arr0.concat(arr1, arr2)
    console.log(arr0) \\[1,2]
    console.log(arr1) \\[3,4]
    console.log(arr2) \\[5,6]
    console.log(arr3) \\[1,2,3,4,5,6]

  • 2.copyWithin, array1.copyWithin(target, start, end)

    1. 复制数组指定位置的内容到数组的另一个指定位置中，会改变现有数组
    2. target:必需，复制到指定目标索引位置
    3. start:可选，复制的起始位置
    4. end:可选，复制的停止位置，负数为倒数,即：length - end
    let arr = [0,2,4,6,8,10]
    arr.copyWithin(3) \\ [0,2,4,0,2,4]
    arr.copyWithin(3,1) \\ [0,2,4,2,4,6]
    arr.copyWithin(3,1,3) \\ [0,2,4,2,4,10]
    arr.copyWithin(3,1,-1) \\ [0,2,4,2,4,6]

  • 3.entries(), array.entries()

    1. 返回键名和键值的Iterator集合
    2. 可以替换 for(let i = 0; i < array.length; i ++) 的写法
    let arr = ['a', 'b', 'c']
    arr.entries()
    [0,'a']
    [1,'b']
    [2,'c']
    for(let [index, val] of arr.entries()){
    console.log(`${index}:${val}`)
    }
    0:a
    1:b
    2:c
